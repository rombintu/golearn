## Обзор
Разработка автоматизированной системы в защищенном исполнении центра повышения квалификации научно-производственного предприятия

## Зависимости 
### Для разработки
* git >= 2.2
* golang >= 1.15
* python >= 3.6
* sqlite3 >= 3.37
* PyQt >= 5.15

Опционально:
* make >= 4.0
* python >= 3.6
* docker >= 20.10
* docker-compose >= 2.2
* postgres >= 12

### Для развёртывания
* docker >= 20.10
* docker-compose >= 2.2
* postgres >= 12

Опционально:
* dpkg >= 1.20
* tar >= 1.30

### Для запуска
* postgres >= 12

## Разработка
```bash
make pre_run
make run_server
```
Go to http://localhost:5000/ping
